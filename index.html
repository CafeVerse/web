✨ Features

🎬

Cinematic Full-Bleed Pages

Beautiful media presentation with smooth gradient banners, live IMDB scores, casting details, and similar recommendation carousels.

⚡

Embedded Media Player

Fully customizable integrated video player with custom styling, subtitle support (.srt & .vtt), and smart state restoration.

📺

Comprehensive Catalog

Seamlessly browse, search, and filter massive movie and television databases powered by high-performance APIs.

⭐

Local Watchlist

A persisted, interactive watchlist built directly into the client with localStorage sync for your viewing preferences.

📂

Auto-Generated Routing

Next.js-style routing with custom file-system routing scanner that converts folder schemas into React Router params.

💻

Cross-Platform Builds

Native application packaging for Windows and Linux with optimized performance and seamless installation.
