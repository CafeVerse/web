📖 Installation Guide

🪟 Windows Installation

Option 1: Using Installer (Recommended)

  1. Download the .exe or .msi installer
  2. Run the installer and follow the on-screen instructions
  3. Launch CaféVerse from your Start Menu or Desktop shortcut

Option 2: Using Portable Version

  1. Download the .zip file
  2. Extract to your desired location
  3. Run CafeVerse.exe directly - no installation required!

🐧 Linux Installation

Using AppImage

chmod +x CafeVerse-*.AppImage
./CafeVerse-*.AppImage

Using Debian Package

sudo dpkg -i cafeverse-*.deb

Troubleshooting: If you encounter permission issues, try:

sudo apt-get install -f
