📖 Installation Guide

Windows

1. Download the .exe or .msi installer
2. Run the installer and follow the on-screen instructions
3. Launch CaféVerse from your Start Menu or Desktop shortcut

Linux

AppImage: Download, make executable, and run directly
chmod +x CafeVerse-*.AppImage && ./CafeVerse-*.AppImage

Debian:
sudo dpkg -i cafeverse-*.deb
